1- Install Linux, preferably UBUNTU
2- install gvim in linux
3- install miniconda:
    goto: https://docs.anaconda.com/miniconda/miniconda-install/
    click: Linux Installer
    click: Download the .sh installer.
    goto Linux table and click "Miniconda3 Linux 64-bit"
    Now you have "Miniconda3-latest-Linux-x86_64.sh" in Download directory
    In Download directory write the following command and install miniconda:
        bash Miniconda3-latest-Linux-x86_64.sh

4- Install pygmt in miniconda:
    open a new terminal and you should have e.g. :
        (base) xxx@xxxxx :~/Download
    where (base) shows that you are in base of miniconda
    follow https://www.pygmt.org/dev/install.html
    click: conda tab
    go to middle of page where you have:
        Installing GMT and other dependencies
            follow the steps as mentioned and I have repeated here:
    In a terminal write:
        conda config --prepend channels conda-forge
        (be patiant for the following command!!)
        conda create --name pygmt python=3.12 numpy pandas xarray netcdf4 packaging gmt
        conda activate pygmt
        conda install pygmt
5- install ipython:
    In a terminal write:
        conda activate pygmt
        conda install -c conda-forge ipython
        
6- test pygmt installation
        further down the same page you see the following, follow steps:
            Testing your install
            but before that you can do the following steps to run the mentioned commands in 
            ipython in a new terminal:
                conda activate pygmt
                ipython
            then in ipython environment you can write the followings as mentioned in testing the install
            ...
            ...
            you should see a map of the earth with PyGMT written on that
